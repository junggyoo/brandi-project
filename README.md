## Features

[서비스]
- 소셜(구글) 로그인 / 로그아웃
- 메인 상품리스트(이미지 슬라이더)
- 상품 구매하기(옵션 선택)
- 상품 결제(배송지 입력)
- 마이페이지(주문 목록)

[어드민]
- 상품 등록(옵션 추가/삭제)
- 상품 관리(검색 필터링)
- 주문 관리
- 주문 상세
- 회원 관리
